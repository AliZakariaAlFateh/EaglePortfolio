افتح المشروع
إزاي تعدّل بياناتكم بسرعة؟
  1. الفريق: افتح index.html وغيّر أوصاف الأعضاء ولينكاتهم داخل Section #team.
  2. المشاريع: كل المشاريع المشتركة داخل Section #projects — انسخ .project-col وزوّد براحتك.
  3. التصنيفات: عدّل data-tags (مثال: web api dashboard) علشان الفلاتر تشتغل.
  4. البحث: بيبحث في اسم المشروع والـ Stack والوصف القصير.
  5. الصور: استبدل assets/img/default-avatar.svg أو غيّر src لكل عضو.
